## 安装
  可以有多种方法进行安装。
### 使用命令行
  github</br>
* Fork 当前库
* 复制一下命令到您的目录</br>
```
https://github.com/hanjiangxueying/vue2-iview2-admin.git
```
## 浏览器支持
* IE 9+
* Firefox（最新）
* Chrome（最新）
* Safari（最新）
## 运行项目
  进入vue2-iview2-admin 运行一下命令安装依赖包。
```
npm install
```
  安装完成之后，开始运行项目。
```
npm run dev 
```
## 编译项目
  运行一下命令：
```
npm run build
```
  自动会在浏览器中打开 http://localhost:8080/#/login 的地址，如 第一幅图所示：</br>
  用户名： 任意输入（还未做校验）</br>
  密码： 大于6个字符。
